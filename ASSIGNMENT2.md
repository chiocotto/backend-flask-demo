# 課題2

**課題1で構築したFlaskの環境を用いて、以下の条件に合致するWeb APIを作成して下さい。**

## 条件

規格：RESTfull API

バリデーション条件：なし

リクエスト形式
```
http://localhost:8080/api/
```

レスポンス形式
```
{
    "message": "Hello World"
}
```
